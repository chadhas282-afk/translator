# 🌐 Modern Language Translator

A clean, responsive, and user-friendly web application that allows users to translate text between dozens of world languages instantly.

## ✨ Features

* **Real-time Translation:** Powered by the MyMemory API.
* **Wide Language Support:** Choose from over 100 languages via the `countries.js` configuration.
* **Text-to-Speech:** Listen to the pronunciation of both the source and translated text.
* **Copy to Clipboard:** Quickly copy translations with a single click.
* **Swap Languages:** Easily switch "From" and "To" languages and their respective text content.
* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Modern UI:** Features a sleek glassmorphism-inspired container and smooth CSS transitions.

## 🚀 Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling with Flexbox and media queries.
* **JavaScript (ES6):** DOM manipulation, Fetch API, and Speech Synthesis.
* **Font Awesome:** Elegant iconography.
* **Google Fonts:** "Poppins" for a modern typography feel.
* **MyMemory API:** External API for language translation.

## 📂 Project Structure

```text
├── index.html      # Main structure of the application
├── style.css       # Visual styling and responsiveness
├── script.js       # Core logic, API calls, and event handling
└── countries.js    # Mapping of country codes to language names

```

## 🛠️ How to Get Started

1. **Clone the repository:**
```bash
git clone https://github.com/chadhas282-afk/translator.git

```


2. **Navigate to the project folder:**
```bash
cd language-translator

```


3. **Run the app:**
Simply open `index.html` in your favorite web browser.

## 🖥️ Usage

1. **Select Languages:** Use the dropdown menus to select the source and target languages.
2. **Input Text:** Type the phrase you want to translate in the left textarea.
3. **Translate:** Click the **"Translate Text"** button.
4. **Listen & Copy:** Use the icons below the text areas to hear the text or copy it to your clipboard.
5. **Exchange:** Click the arrows icon in the center to swap the languages and text.
